# BlogsHQ

## Overview

**Blog Web Application** built with **React**, **Redux**, and **Ant Design**.  
It enables users to create, read, update, and delete blog articles with a modern, responsive UI. The app integrates **Formik** for form management, **Yup** for validation, and **Firebase** for backend data storage.

The project uses **TypeScript** for type safety, **Webpack** for bundling, and is configured with **ESLint**, **Prettier**, **Husky**, and **lint-staged** to maintain high code quality.

---

## Features

- Full CRUD operations for blog articles
- Form validation with **Formik** & **Yup**
- Modern UI using **Ant Design**
- State management with **Redux**
- Client-side routing with **React Router DOM**
- Firebase backend integration
- Environment configuration via **dotenv**
- Responsive and mobile-friendly design
- TypeScript support
- Pre-commit hooks to ensure code quality

---

## Tech Stack

- **Frontend:** React 19, Redux, React Router DOM
- **UI Library:** Ant Design
- **State Management:** Redux
- **Forms & Validation:** Formik & Yup
- **Bundler:** Webpack
- **Language:** TypeScript & JavaScript
- **Styles:** CSS, SCSS, Less
- **Backend / Database:** Firebase
- **Environment Management:** dotenv
- **Code Quality Tools:** ESLint, Prettier, Husky
